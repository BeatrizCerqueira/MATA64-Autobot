/*
*	Configurando jFuzzyLogic
*/
- Baixe o arquivo .jar disponível em:
	https://sourceforge.net/projects/jfuzzylogic/files/latest/download
- Mova o arquivo baixado (jFuzzyLogic.jar) para C:\robocode\libs
- Edite o arquivo C:\robocode\robocode.bat adicionando ao parametro cp (classpath) o atual caminho do arquivo baixado, a saber libs/jFuzzyLogic.jar
	Exemplo:
		ANTES
			java -DNOSECURITY=true [...] -cp libs/robocode.jar;C:/swipl/bin/jpl.dll;C:/swipl/lib/jpl.jar;C:/swipl/library/jpl.pl;C:/swipl/bin/libpl.dll [...]
		DEPOIS
			java -DNOSECURITY=true [...] -cp libs/robocode.jar;C:/swipl/bin/jpl.dll;C:/swipl/lib/jpl.jar;C:/swipl/library/jpl.pl;C:/swipl/bin/libpl.dll;libs/jFuzzyLogic.jar [...]
- Basta abrir o robocode e quando quiser importar as classes do jFuzzyLogic utilizar:
	import net.sourceforge.jFuzzyLogic.FIS;
	import net.sourceforge.jFuzzyLogic.FunctionBlock;
	import net.sourceforge.jFuzzyLogic.plot.JFuzzyChart;
	...



/*
*	Configurando jBayes
*/

- Baixe o zip do repositorio do jAyes disponível em:
	https://codeload.github.com/kutschkem/Jayes/zip/refs/heads/master
- Depois de  descompactar, mova a pasta org contida em Jayes-master\plugins\org.eclipse.recommenders.jayes\src\org para C:\robocode\robots
- Basta abrir o robocode e quando quiser importar as classes do jFuzzyLogic utilizar:
	//	--------------	IMPORTACAO BAYES
	import org.eclipse.recommenders.jayes.BayesNet;
	import org.eclipse.recommenders.jayes.BayesNode;
	import org.eclipse.recommenders.jayes.inference.IBayesInferer;
	import org.eclipse.recommenders.jayes.inference.junctionTree.JunctionTreeAlgorithm;

	//	--------------	IMPORTACOES DE UTILITARIOS
	import java.util.ArrayList;
	import java.util.Collections;
	import java.util.HashMap;
	import java.util.List;
	import java.util.Map;
	import java.util.Arrays;
